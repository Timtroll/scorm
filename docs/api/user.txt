## ЗАПРОСЫ

1. Method POST `/user/add_by_email` - Добавить запись о пользователе в таблицу

запрос:

| Поле | Тип | Описание | Значение |
|---|---|---|---|
| surname | text | Фамилия | До 24 букв, обязательное поле |
| name | text | Имя | До 24 букв, обязательное поле |
| patronymic | text | Отчество | До 32 букв, обязательное поле |
| place | text | Адрес | До 64 букв, знаков или цифр, обязательное поле |
| country | select | Страна | До 32 букв, обязательное поле |
| timezone | select | Часовой пояс | До 8 цифр с плюсом или минусом, обязательное поле |
| birthday | text | Дата рождения | До 12 цифр, обязательное поле |
| password | text | Пароль | До 32 символов, обязательное поле |
| avatar | text | Id загруженного аватара | До 64 символов, обязательное поле |
| type | select | Тип | 1 цифра, обязательное поле |
| email | text | Электронный адрес ||

ответ:
```
{
  "publish": "ok",
  "id": "123"
}

{
  "publish": "fail",
  "message":"причина ошибки"
}
```
